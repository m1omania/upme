# Описание функциональных возможностей приложения UpMe

## Общее описание

UpMe — веб-приложение для автоматизации поиска работы на HeadHunter.ru. Помогает соискателям находить релевантные вакансии, генерировать персонализированные сопроводительные письма с помощью ИИ и отправлять отклики.

## Основные функции

1. **Авторизация через OAuth 2.0** — вход через HH.ru, управление токенами доступа
2. **Управление резюме** — загрузка и отображение опубликованных резюме из HH.ru
3. **Поиск вакансий** — поиск с фильтрами (опыт, локация, зарплата), расчет релевантности, исключение уже отправленных откликов
4. **Генерация писем ИИ** — автоматическое создание персонализированных сопроводительных писем
5. **Отправка откликов** — автоматическая отправка откликов через HH.ru API
6. **Управление откликами** — история и статистика отправленных откликов
7. **Профиль пользователя** — информация из HH.ru, настройка фильтров поиска

## Используемые методы HH.ru API

- **OAuth 2.0**: `GET /oauth/authorize`, `POST /oauth/token` (получение и обновление токенов)
- **Пользователь**: `GET /me` (информация о пользователе)
- **Резюме**: `GET /resumes/mine`, `GET /resumes/{resume_id}`
- **Вакансии**: `GET /vacancies`, `GET /vacancies/{vacancy_id}`
- **Отклики**: `POST /negotiations` (создание отклика с резюме и сопроводительным письмом)
